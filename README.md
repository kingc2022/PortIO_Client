# PortIO Client

## Getting started

1. Install Python Interpreter **(< 3.11)**
2. Generate a virtual environment
```shell
python -m venv venv
```
3. Change into the virtual environment
4. Install dependencies
```shell
pip install -r requirements.txt
```

## 奇怪的命名

- `nolx`，如 `nol1`：`No Operation Label` 即无需操作的标签
